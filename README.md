機能要件
- 8時間に一回ステッピングモーターを1回転させる
- YouTubeの特定アカウントのライブでスパチャが来たときにステッピングモーターを1回転させ、30秒後に定型文をチャットに投稿する
- Web上に配置したボタンを押したときにステッピングモーターを1回転させる

機能ごとに処理を細分化
- ステッピングモーターを回す
- タイマ割り込み
- YouTubeのスパチャを監視
- YouTubeに定型チャットを送る
- Webサーバを起動する
  - ボタンを表示
  - POST時の処理
